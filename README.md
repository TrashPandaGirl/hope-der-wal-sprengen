# Timmy den Wal sprengen 💥

Ein absurdes Browser-Klickerspiel. Timmy der Wal schwimmt durchs Bild — deine Aufgabe: ihn sprengen.

**[▶ Jetzt spielen](https://trashpandagirl.github.io/hope-der-wal-sprengen/timmy.html)**

## Gameplay

- Lade die Bombe auf, indem du die Maustaste gedrückt hältst
- Bei 100% Ladung: Klick → Explosion
- Sammle Münzen 🪙 und kaufe Upgrades im Shop

## Shop

| Item | Effekt |
|------|--------|
| 🚧 Bauzaun | Blockiert Timmys Weg |
| 🎩 Politiker | Schickt Heiler weg |

## Tech

Single-file HTML/JS/CSS — kein Build, kein Framework, läuft direkt im Browser.
